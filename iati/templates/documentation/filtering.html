Interface with the OIPA resources

Search

The current API implements a combined search query on the title and description fields of the ActivityResource.

http://127.0.0.1:8080/api/v2/activities/?format=json&query=Food

Note: The search is case insensitive

Countries

The current API implements a countries filter

http://127.0.0.1:8080/api/v2/activities/?format=json&countries=BB|AO|FR

Joining calls

Calls to the API can be joined

http://127.0.0.1:8080/api/v2/activities/?format=json&query=Food&countries=BB|AO|FR
